# Customer Accounts Microservice

![Build Status] (https://github.com/IllaMANASA/customers-account/actions/workflows/ci-build.yaml/badge.svg)

## Description
This project is a RESTful microservice for managing customer accounts.

## Features
- Create Account
- Read Account
- Update Account
- Delete Account
- List Accounts

## Tech Stack
- Python (Flask)
- Docker
- Kubernetes
- GitHub Actions
