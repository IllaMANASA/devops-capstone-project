# User Stories Template

## Format

As a <role>  
I need <function>  
So that <benefit>  

---

## Example User Stories

1. As a customer  
   I need to create an account  
   So that I can access services  

2. As a customer  
   I need to view my account  
   So that I can check my details  

3. As a customer  
   I need to update my account  
   So that my information is correct  

4. As a customer  
   I need to delete my account  
   So that I can remove my data
